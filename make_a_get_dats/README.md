# YouTube Data Fetcher
A simple Python script to fetch data from YouTube videos or channels.

## Requirements
* Python 3.x
* pytube library (install with `pip install pytube`)

## Usage
1. Run the script with `python main.py`.
2. Enter the YouTube video or channel URL when prompted.
3. The script will print the fetched data to the console.

## Note
This script uses the pytube library, which does not support getting channel data directly.
For channel data, you may need to use the YouTube API.
